SISTEM MANAJEMEN FILM / LIST MOVIE
-----------------------------------

Tugas Akhir Praktikum Algoritma dan Struktur data

-------
 FITUR
-------
- Sign Up
- Login / Sign In
  
- Fitur Khusus User
  1. User bisa melihat daftar/list Film yang suda ada;
  2. Cari film berdasarkan judul dan genre menggunakan logika Saerching (Binary);
  3. Rekomendasi film : Dipilih dari genre (Pakai Searching -Binary-) dan diurutkan dari rating tertinggi (Pakai Sorting -Quick Sort-);
  4. Edit watchlist (Pakai Single Linked List), misal tambah dan hapus witchlist;
  5. Navigasi film (Pakai Double Linked List), misal next film, previous film;
  6. Riwayat tontonan (Pakai File + Single Linked List), disimpan di File history.txt.

-Fitur Khusus Admin
  1. Tambah film : Pakai Head Tail Linked List (Judul, Genre, Rating) yang disimpan di FILE;
  2. Lihat Film : Pakai Sorting ascending (Bubble Sort), descending (Quick Sort);
  3. Edit Film;
  4. Hapus film.
 
- Logout
